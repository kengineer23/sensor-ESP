Implementation of the PMS5003Sensor class for interfacing with the PMS5003 sensor.

This file contains the implementation of the PMS5003Sensor class, which provides methods for initializing and reading data from the PMS5003 particulate matter sensor. The class uses the Serial2 interface on the ESP32 for communication with the sensor.
