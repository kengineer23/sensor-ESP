Detailed Description

A structure to hold data from various sensors.

This structure contains data from multiple sensors, including the DHT11 sensor, PMS5003 particulate matter sensor, and the MQ7 gas sensor. It provides a convenient way to manage and access the data from these sensors as a single unit.
