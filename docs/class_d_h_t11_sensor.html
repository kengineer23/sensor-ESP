Detailed Description

A class for interfacing with the DHT11 temperature and humidity sensor.

The DHT11Sensor class provides methods to initialize and read data from the DHT11 sensor. It uses the Adafruit DHT library to interface with the sensor and retrieve temperature and humidity readings.
